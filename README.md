Eternal Edge Defense: Chain Graph Simulator

An interactive, turn-based graph theory game demonstrating the Eternal Relaxed Vertex Cover (ERVC) problem on complex, multi-tiered Chain Graphs.

📖 The Mathematics

In static graph theory, defending a graph requires placing guards on a Vertex Cover (a set of nodes that touches every edge). However, in the Eternal Edge Defense model, guards are highly mobile and can slide along edges to respond to attacks dynamically.

This simulator models a 3-tier Chain Graph (nested bipartite graph).

Static Vertex Cover: Requires 7 guards glued to the floor to cover every edge.

Eternal Relaxed Vertex Cover (ERVC): Requires only 3 guards if the defender utilizes a perfect "Hub & Floater" strategy to cycle guards between nested neighborhoods.

🎮 How to Play

Phase 1: Setup

Click on nodes to deploy your guards.

Try to find the optimal mathematical minimum (Hint: try placing 3 guards on the super-hubs).

Click Lock Guards & Start Game when ready.

Phase 2: Attacker Turn

The attacker (you or a friend) clicks any edge to attack it.

If a guard is already on one of the endpoints, the attack is automatically deflected!

Phase 3: Defender Turn

If an edge is undefended, the defender must slide a guard to cover it.

Click one of your guards, then click an adjacent empty node that connects to the attacked edge.

Beware the Death Trap: If the attacker forces your guards into the "dead-end" clusters of the chain graph, they can snap your defense line!
